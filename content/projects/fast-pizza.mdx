# Fast Pizza 🍕

Fast Pizza is a modern frontend pizza ordering interface built with **React** and **TailwindCSS**. Users can explore the menu, customize pizzas, and see a dynamic order preview. This project demonstrates interactive UI design and responsive layout without backend integration.

## Core Features

### 1. Menu System

- Display pizza items dynamically
- Ingredient customization
- Real-time price calculation

### 2. Order Preview

- Add/remove items to/from cart
- See live cart total
- Responsive design for desktop and mobile

### 3. UI/UX Design

- Built with **TailwindCSS** for modern, responsive layouts
- Interactive components for a smooth user experience
- Clean and maintainable React component structure

## Technology Stack

- **React 18** (Vite)
- **Tailwind CSS**
- Component-driven UI design
