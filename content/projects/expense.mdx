# Expense Tracker 📊

A **Flutter** mobile application that allows users to manage their expenses efficiently. The app supports full **CRUD operations** for expense entries, including adding, updating, deleting, and viewing records.

## Core Features

### 1. Expense Management

- Add, edit, and delete expenses
- Categorize expenses (e.g., Food, Transport, Utilities)
- View expense history by day, week, or month

### 2. User Experience

- Intuitive and interactive UI with Flutter widgets
- Responsive design for different mobile screen sizes

### 3. Analytics

- Display total expenses by category
- Track spending trends over time

## Technology Stack

- **Flutter**: App frontend and logic
- **Dart**: Programming language
