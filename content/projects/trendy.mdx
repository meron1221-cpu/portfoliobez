Trendy is a modern **eCommerce platform** designed to provide a smooth shopping experience. The project is powered by **React** on the frontend with **Redux Toolkit (RTK)** for state management, while the backend is handled by **Laravel**. For styling, it uses **Tailwind CSS** along with the **Tailwick theme** to ensure a sleek and responsive UI.

## Features

- **Product Catalog**: Browse products with filtering and sorting.
- **Cart & Checkout**: Add products to the cart and proceed with secure checkout.
- **Orders Management**: Track user orders and manage order history.
- **Authentication**: Secure user login and registration system.
- **Admin Dashboard**: Manage products, orders, and settings.

## Technologies

- **React.js**: Frontend library for building interactive UIs.
- **Redux Toolkit (RTK)**: State management for cart, user, and orders.
- **Laravel**: Backend framework for API development.
- **Tailwind CSS**: Utility-first CSS framework for responsive design.
- **Tailwick**: Prebuilt theme for enhanced UI/UX.
