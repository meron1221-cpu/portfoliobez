ShamSuper Store is a complete eCommerce solution built with **React** and **Laravel**. It provides a seamless shopping experience for customers while giving full control to the admin to manage products, orders, deliveries, and payments.  

## Features

- **Full Order System**: Customers can browse products, add to cart, place orders, and track their status.  
- **Delivery Tracking**: Integrated order tracking system with live location updates.  
- **Dynamic Payment Methods**: Users can add or choose different payment methods.  
- **Admin Dashboard**: Manage products, categories, orders, users, and delivery status.  
- **Customer Dashboard**: Track orders, update profiles, and manage payments.  
- **Secure Authentication**: JWT-based authentication with role-based access.  

## Technologies

- **React.js**: Frontend framework for building interactive UI.  
- **Laravel**: Backend framework for API and business logic.  
- **Redux Toolkit (RTK)**: State management for the frontend.  
- **Tailwind CSS + Tailwick Theme**: Modern UI styling.  
- **MySQL**: Database for managing users, orders, and products.
