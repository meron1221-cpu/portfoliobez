# Afrobillboard

Afrobillboard is a marketplace platform similar to *Made-in-China*, built to showcase African products and connect sellers with buyers globally. The platform eliminates the need for individual seller dashboards—product listings and negotiations are fully managed by the company.

## Project Structure

The application follows a **React + Laravel** architecture:

- **Frontend**: React (for dynamic UI)  
- **Backend**: Laravel (API + business logic)  
- **Database**: MySQL  
- **Authentication**: Laravel Socialite for Google login + custom authentication  
- **State Management**: Redux Toolkit  

## Features

### Marketplace Management
- **Centralized Product Listings**: Products posted under company management, no separate seller pages  
- **Influencer-Driven Marketing**: Products linked to influencers for visibility  
- **Company-Negotiated Deals**: Buyers interact with the company directly  

### User Experience
- **Guest Checkout**: Purchase without account creation  
- **Smart Search & Filters**: Discover products by category, influencer, or region  
- **Multilingual Support**: English + Portuguese (in progress)  

### Admin Features
- **Product & Order Management**  
- **Payment Tracking**  
- **Coupon & Discounts System**  
- **Reviews & Feedback**  

## Technology Stack

### Frontend
- **React**: Component-based UI  
- **Redux Toolkit**: State management  
- **React Router**: Navigation  
- **Axios**: API requests  

### Backend
- **Laravel 10**: REST API with business logic  
- **Sanctum**: Authentication tokens  
- **Socialite**: Google authentication  
- **MySQL**: Relational database  
- **Redis**: Queue management
